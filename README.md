# Lista de usuários

Aplicação web desenvolvida com Angular 

## Funcionalidades
- Listagem dos usuários 
- Aplicação de filtro para encontrar algum usuário

## Conceitos abordados
- Property Binding
- Event Binding
- Two Way Data Binding
- Decorator @Input e @Output
- Pipe
- Componentes

## Bibliotecas
- date-fns
